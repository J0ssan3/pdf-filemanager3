
# Projeto JavaScript Manual: PDF File Manager

Este projeto permite extrair texto de arquivos PDF sem npm, Maven ou Gradle.

## Como usar:

1. Abra o arquivo `index.html` no navegador.
2. Selecione um PDF.
3. Clique em "Extrair".
4. O texto será salvo como `saida.txt`.

## Observação:
Este exemplo usa bibliotecas JS como PDFLib e FileSaver carregadas localmente.
